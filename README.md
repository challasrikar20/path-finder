# path-finder
**Graph Algorithm Wizard** 🚀
An interactive web application for visualizing graph algorithms and finding the shortest path using Dijkstra's Algorithm. Built with Python, Flask, NetworkX, Matplotlib, HTML, CSS, and JavaScript, this project helps students and developers understand graph theory through an intuitive step-by-step interface.

**✨ Features**
🎯 Interactive Graph Wizard UI
📊 Supports both Weighted and Unweighted graphs
🛣️ Finds the shortest path using Dijkstra's Algorithm
🔴 Highlights the shortest path visually
🎨 Multiple graph layouts:
Spring Layout
Circular Layout
Random Layout
Planar Layout
🌐 Web-based interface built with Flask
📈 Displays graph edge weights and total shortest distance
📚 Educational tool for learning Graph Theory and Data Structures
**🛠️ Tech Stack**

Python
Flask
NetworkX
Matplotlib
HTML5
CSS3
JavaScript
**📂 Project Workflow**

Select the application mode.
Enter graph details (vertices, edges, and weights).
Choose graph type (Weighted/Unweighted).
Select a graph layout.
Run Dijkstra's Algorithm.
View the generated graph with the shortest path highlighted in red.
